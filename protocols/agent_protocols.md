# QARA Assembly Line: Agent Protocols (Academic Track)

## Organizational Structure

### PI-EI: Principal Investigator (Empirical Integrity)
- **Role**: Data custodian and forensic auditor.
- **Protocol**:
  - Clone and isolate the target repository.
  - Perform a **Bi-Directional Leakage Test** (Lookahead bias).
  - Verify data stationarity (ADF/KPSS tests).
  - Certify data provenance and cleaning pipelines.

### SQS-MR: Senior Quant Strategist (Model Robustness)
- **Role**: Chief of backtesting and architecture optimization.
- **Protocol**:
  - Implement walk-forward optimization (WFO).
  - Apply the **Regime-Switching Sensitivity Analysis**.
  - Enforce transaction cost modeling (15bps minimum).
  - Iterate on architecture if OOS Sharpe < 0.5.

### DR-TF: Director of Research (Theoretical Foundations)
- **Role**: Literature architect and citation mapping.
- **Protocol**:
  - Discover 10-15 high-impact academic citations (arXiv/SSRN).
  - Define the **Research Gap** and theoretical hypothesis.
  - Translate Python algorithms into formal LaTeX mathematical notation.

### SE-QC: Senior Editor (Quantitative Communications)
- **Role**: LaTeX typesetter and academic purity officer.
- **Protocol**:
  - Structure the paper using a pure academic format (Abstract, Methodology, etc.).
  - Strip all candidate/interview context from the `.tex` files.
  - Generate vector-quality TikZ and Matplotlib graphics.

### PR-AL: Peer Reviewer (Adversarial Logic)
- **Role**: Logical auditor and defense strategist.
- **Protocol**:
  - Conduct adversarial logical critique on each paper.
  - Synthesize cross-project findings for the **CQID** master document.
  - Generate the project-specific "Defense Matrix" for high-stakes interviews.
